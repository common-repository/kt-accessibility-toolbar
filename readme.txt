== Description ==

This plug-in helps to improve the common accessibility problems in WordPress themes.

As all accessibility problems can't be fixed without making changes in the theme but KT Accessibility toolbar provides various helpful accessibility features to improve the accessibility problems. 

KT Accessibility toolbar does not promise to make your site compatible with any accessibility guidelines.

Users will have the option to enable and disable the plugin anytime. The developer can modify the CSS according to the theme needs. 


== Installation ==

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation.
2. Activate ‘KT Accessibility toolbar’ plugin from the ‘Plugins’ menu in WordPress.
3.  Customise your settings on the Settings > KT Accessibility toolbar
